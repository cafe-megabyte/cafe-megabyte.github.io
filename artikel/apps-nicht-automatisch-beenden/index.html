Apps nicht automatisch beenden

Grundsätzliches

Mac OS beendet viele Programme nachdem das letzte Fenster geschlossen wurde.

Vorgehen

Automatisches Beenden abschalten:

defaults write -g NSDisableAutomaticTermination -bool yes

Standardverhalten wiederherstellen:

defaults delete NSDisableAutomaticTermination
