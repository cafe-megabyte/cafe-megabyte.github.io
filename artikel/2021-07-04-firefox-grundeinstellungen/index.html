Firefox Grundeinstellungen

Allgemein

  • Sitzung nicht automatisch wiederherstellen
  • “Links in Tabs statt Fenster” aktivieren, alle anderen bei “Tab” deaktivieren
  • Vor Update fragen
  • “Automatischen Bildlauf” deaktivieren

Startseite

  • 2x leere Seite für Fenster und Tabs

Suche

  • Getrenntes Suchfeld aktivieren
  • Standardsuche “DuckDuckGo”
  • Alle Suchmaschinen-Schlüsselwörter deaktivieren
  • Sämtliche Vorschläge im Adressfeld deaktivieren

Datenschutz & Sicherheit

  • Standard-Einstellungen verwenden
  • Cookie und Website-Daten beim Beenden löschen
  • PW-Speicherung aktivieren
  • Automatisches Ausfüllen aktivieren
  • Chronik anlegen, jedoch beim Beenden löschen lassen:
    • Cookies
    • Cache
    • Aktive Logins
    • Offline-Website-Daten

about:config

  • Suchmaschine deaktivieren: keyword.enabled ==> false
  • Automatische Ergänzung von “.com” in der URL verbieten: browser.fixup.alternate.enabled => false
  • Nativen Video-Vollbildmodus aktivieren: full-screen-api.macos-native-full-screen ==> true
  • Import von Login-Daten aus einer CSV-Datei erlauben: signon.management.page.fileImport.enabled ==> true
  • Kompakte Darstellung aktivieren: browser.uidensity ==> 1
  • Darstellung von Umlautdomains verbieten: network.IDN_show_punycode ==> true
  • Download-Popup verhindern, damit Downloads ohne Rückfrage starten: browser.download.improvements_to_download_panel ==> true
  • Hell-/Dunkel-Modus entsprechend der Systemeinstellung anstatt nach der Farbe des Theme-Bildes: layout.css.prefers-color-scheme.content-override ==> 2
  • Keine Warnung bei “unsicheren” Downloads per http: dom.block_download_insecure ==> false
  • Webseite beim Auffrischen immer vollständig neu laden: browser.cache.check_doc_frequency ==> 1

Autoconfig

Durch einen Bug in Firefox wird beim Wechsel des Erscheinungsbildes in den Systemeinstellungen (hell <> dunkel), eine Einstellung geändert, sodaß sich Firefox-Fenster (z.B. Verlauf oder Lesezeichen) nicht mehr nach den Systemeinstellungen, sondern an dem aktuell eingestellten Theme orientieren. Um das zu verhindern, muß die enstprechende Einstellung gegen Änderungen gesperrt werden.

Hierzu sind 2 Textdateien erforderlich, die in den Firefox-Programmordner kopiert werden müssen:

autoconfig.js muß abgelegt werden in Firefox.app/Contents/Resources/defaults/pref/:

pref("general.config.filename", "firefox.cfg");
pref("general.config.obscure_value", 0);

firefox.cfg muß abgelegt werden in Firefox.app/Contents/Resources/:

// IMPORTANT: Start your code on the 2nd line
lockPref("browser.theme.toolbar-theme", 2);
