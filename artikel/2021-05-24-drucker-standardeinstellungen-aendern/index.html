Drucker Standardeinstellungen ändern

Web-Interface aktivieren:

cupsctl WebInterface=yes

URL öffnen:

http://localhost:631/printers

Über das Menü “Administration > Set Default Settings” können die Vorgaben geändert werden.

Anschließend das Web-Interface wieder deaktivieren:

cupsctl WebInterface=no
