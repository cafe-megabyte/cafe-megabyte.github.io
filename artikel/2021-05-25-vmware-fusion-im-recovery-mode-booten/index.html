VMware Fusion im Recovery Mode booten

Recovery Mode erzwingen

VM ausschalten.

Konfigdatei editieren (Option + rechte Maus auf die VM), dort eine Zeile einfügen:

macosguest.forceRecoveryModeInstall = "TRUE"

VM starten.

Zurück zum normalen System

VM ausschalten.

Diese eingefügte Zeile wieder aus der Konfiguration löschen.

Innerhalb der virtuellen Maschine im Finder die Datei mit der Ednung .nvram löschen.

VM starten.
